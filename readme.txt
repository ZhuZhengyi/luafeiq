luaFeiq is a simple IP messager lua implementation. It's main goal is to 
communicate with Windows FeiQ. Because it's totally written by Lua, so 
it 's crossplatform.

Just check out doc/install_dev.txt file to install run enviroment and:
cd src
lua ui_main.lua 
to run luaFeiq.
